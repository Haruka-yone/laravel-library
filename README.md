# 📚 Laravel Library

Laravel × Bootstrap で開発した著者管理機能付きの図書管理アプリです。  
ログイン認証、著者・書籍のCRUD、プロフィール管理など、Webアプリの基本機能を実装しています。

### ✨ Features

- 🔐 Laravel UI によるログイン認証  
- ✍️ 著者の入力 / 編集 / 削除  
- 📘 本の登録 / 写真登録 / 編集 / 削除  
- 🙍‍♂️ プロフィール編集  
- 🗂 投稿一覧ページ（自分の投稿管理）

### 🔧 Tech Stack

- Laravel  
- PHP  
- Bootstrap
